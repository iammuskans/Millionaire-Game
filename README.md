# Millionaire Game

A simple command-line based Millionaire quiz game, built using Python.

# Features

* Multiple-choice questions
* Prize-based progression system 💰
* Instant feedback on answers
* Game ends on wrong answer

# Tech Used

* Python
* Basic concepts: loops, lists, conditionals, user input
